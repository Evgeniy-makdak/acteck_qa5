# Acteck QA5 (MT5 Port)

Полный перенос проекта `acteck-qa` с MQL4 на MQL5 с сохранением основной логики расчёта трендов, вероятностей и рабочего процесса анализа.

**Автор стратегии и проекта:** Evgeniy Acteck  
**Порт на MT5:** выполнен для репозитория `acteck_qa5`

---

## Что это за советник

`Acteck.mq5` — аналитический советник/индикаторный инструмент для MetaTrader 5, который:

- анализирует исторические участки направленного движения;
- определяет развороты по заданному фильтру волатильности (в пунктах);
- строит трендовые отрезки на графике;
- рассчитывает вероятностные уровни продолжения/разворота;
- показывает рабочую таблицу по набору символов и фильтров;
- формирует CSV-отчёты в `MQL5/Files/Acteck/`.

Логика взята из MT4-версии (`Acteck.mq4`) и сохранена максимально близко:

- те же ключевые входные параметры;
- те же принципы `GetDirection / CheckUp / CheckDn`;
- тот же подход к отчётам и ранжированию вероятностей;
- тот же подход загрузки символов из `.set` файла.

## Куда копировать файлы в MT5

### 1) Открыть папку данных MT5

В терминале MetaTrader 5:

`File -> Open Data Folder`

### 2) Скопировать советник

Из проекта:

- `MQL5/Experts/Acteck/Acteck.mq5`

В папку терминала:

- `MQL5/Experts/Acteck/Acteck.mq5`

Если подпапки `Acteck` нет — создать.

### 3) Скопировать файл символов (ВАЖНО)

Из проекта:

- `MQL5/Files/Acteck/forex.set`

В папку терминала:

- `MQL5/Files/Acteck/forex.set`

Именно эта локация используется в коде:

- `Acteck\\<NameSet>.set`

По умолчанию `NameSet=forex`, значит советник ищет:

- `MQL5/Files/Acteck/forex.set`

## Компиляция

1. Открыть `Acteck.mq5` в MetaEditor.
2. Нажать `Compile (F7)`.
3. Получить `Acteck.ex5`.

Если компиляция успешна, файл появится в:

- `MQL5/Experts/Acteck/Acteck.ex5`

---

## Запуск на графике

1. Открыть любой график в MT5.
2. Перетащить `Acteck` из `Navigator -> Experts`.
3. Разрешить AutoTrading (если терминал спросит).
4. Убедиться, что в `NameSet` задано имя существующего файла (`forex` по умолчанию).

## Основные входные параметры

- `RepType` — авто/ручной выбор символа для отчётов.
- `Symb` — символ для ручного отчёта.
- `Timeframe` — ТФ отчёта.
- `StartDate / EndDate` — интервал исторического анализа.
- `UseCurrentEndDate` — если `true` (по умолчанию), конечная дата автоматически равна текущей дате.
- `MinPips` — базовый фильтр волатильности.
- `NameSet` — имя файла символов (без `.set`).
- `Level1..Level5` + `Timeframe1..Timeframe5` — столбцы таблицы.
- `Updater` — частота обновления, сек.
- `ATRPeriod` — период ATR для колонки ATR.
- `EnableAlerts` — звуковой сигнал при вероятности >= 60%.
- `ShowOnlyCurrentSymbol` — если `true` (по умолчанию), таблица показывает только 1 строку для текущего символа графика (быстрее и легче для терминала).

## Логика работы (коротко)

1. Советник загружает список инструментов из `MQL5/Files/Acteck/forex.set`.
2. Для каждого инструмента и каждого фильтра:
   - строит последовательность трендов;
   - оценивает вероятность по статистике прошлых движений;
   - считает коррекцию и текущее отклонение.
3. В таблице окрашивает ячейки:
   - синий — направление вверх (buy);
   - красный — направление вниз (sell);
   - белый — нейтрально.
4. По клику на ячейку:
   - переключает график на выбранный символ/таймфрейм;
   - рисует разметку и вероятностные уровни.
5. Пишет CSV-отчёты:
   - `MQL5/Files/Acteck/Table_1_<SYMBOL>.csv`
   - `MQL5/Files/Acteck/Table_2_<SYMBOL>.csv`

## Важные замечания по соответствию MT4 -> MT5

- Портирован расчётный алгоритм, отчёты и общая логика выбора/отображения.
- GUI реализован штатными объектами MT5 (`OBJ_LABEL`, `OBJ_BUTTON`, `OBJ_TREND`) без внешней MT4-библиотеки, чтобы проект был самодостаточным в MT5.
- Формат `forex.set` сохранён построчным списком символов.
- Добавлено авто-сопоставление символов с брокерскими суффиксами (например `EURUSD` -> `EURUSDpf`).
- Вероятностные линии выводятся с явными текстовыми подписями на графике.

## Рекомендации по торговому применению

Инструмент используется как аналитический модуль принятия решений:

- сначала фильтрация пар по вероятности;
- затем сверка уровней, коррекции и текущего отклонения;
- вход в сделку подтверждается рыночным контекстом (уровни/новости/ATR).

Согласно исходной методологии Evgeniy Acteck:

- не входить в рынок без сигнала;
- работать от ложного пробоя и разворотной логики;
- соблюдать риск-менеджмент и целевое соотношение риска к прибыли.
